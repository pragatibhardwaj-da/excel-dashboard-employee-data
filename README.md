# Excel Employee-Info Dashboard

## 📊 Project Overview
This project analyzes Employee data using an interactive Excel dashboard.

##  Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

##  Key Insights
- Performance of departments
- Hiring trend over years
- Employees per department
- Gender distribution
